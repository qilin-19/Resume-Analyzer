# 各岗位常见技能要求参考

> 这份资料帮助 LLM 在分析 JD 时，更准确地识别技能要求。
> 你可以随时编辑——加新岗位、改要求，下次分析就会参考新内容。

---

## 前端开发
### 必备
- HTML, CSS, JavaScript (ES6+)
- React / Vue / Angular 至少精通一个
- Git 版本控制

### 加分
- TypeScript
- Next.js / Nuxt (SSR 框架)
- Tailwind CSS / Sass / CSS-in-JS
- 前端性能优化（Lighthouse、Core Web Vitals）
- 单元测试（Jest、Vitest）+ E2E 测试（Playwright、Cypress）
- Webpack / Vite 配置经验


### 了解
- Node.js 基础、REST API 设计
- Docker 基础使用
- CI/CD 基本概念

---

## 后端开发
### 必备
- Python / Java / Go / Node.js 至少精通一个
- SQL 与关系型数据库（MySQL / PostgreSQL）
- RESTful API 设计

### 加分
- Docker + Kubernetes
- 微服务架构
- 消息队列（Kafka、RabbitMQ、Redis Stream）
- NoSQL（MongoDB、Redis、Elasticsearch）
- 系统设计能力

### 了解
- CI/CD（Jenkins、GitHub Actions）
- 云服务（AWS / Azure / GCP）
- gRPC、GraphQL

---

## AI / 机器学习工程师
### 必备
- Python
- PyTorch / TensorFlow 至少一个
- 数据处理（Pandas、NumPy）
- 机器学习基础（分类、回归、聚类）
- agent

### 加分
- LLM 应用开发（LangChain、Agent、RAG）
- 大模型微调（LoRA、QLoRA）
- 模型部署（Triton、vLLM、Ollama）
- MLOps（MLflow、Weights & Biases）
- NLP / CV / 推荐系统专长
- agent
- claude code


### 了解
- 分布式训练（DeepSpeed、FSDP）
- 模型压缩（量化、剪枝、蒸馏）
- 强化学习

---

## 全栈开发
### 必备
- 前后端各会一门主流语言/框架
- 数据库设计与优化
- Git + CI/CD

### 加分
- DevOps 能力（Docker、K8s、Terraform）
- 系统架构设计
- 项目管理经验
- 跨平台开发（React Native、Flutter、Electron）

---

## 移动端开发
### 必备
- React Native / Flutter / Swift / Kotlin 至少一个
- 移动端 UI 适配
- 状态管理（Redux、MobX、Riverpod）

### 加分
- 原生开发经验
- App 性能优化
- 推送、支付、地图等 SDK 集成
- App Store / Google Play 发布经验

---

## DevOps / SRE
### 必备
- Linux 系统管理
- Docker + Kubernetes
- CI/CD 流水线搭建

### 加分
- Terraform / Ansible (IaC)
- 监控告警（Prometheus、Grafana）
- 日志系统（ELK、Loki）
- 云服务深度使用（AWS / Azure / GCP）
